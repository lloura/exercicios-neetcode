# Exercícios NeetCode

Repositório com minhas soluções dos exercícios do [NeetCode](https://neetcode.io/) implementadas em C++.

Criei isso para praticar algoritmos e estruturas de dados, e decidi compartilhar com a intenção de que talvez possa servir de referência para mim e/ou para outras pessoas no futuro.

---

## Sobre

Implementei as soluções em C++ conforme resolvia os exercícios. Cada arquivo está repleto de comentários explicando o passo a passo da lógica e a abordagem utilizada, para fins de clareza e legibilidade do código.

---

## Compilando

### Meu setup

Usei a extensão **Code Runner** do VSCodium, que compila com g++ e executa com um clique no botão de play.

### Outras opções

Os códigos devem compilar normalmente em outras IDEs: Dev-C++, CLion, Code::Blocks, VS Code, etc.

### Compilação manual

```bash
cd "arrays & hashing"
g++ containsDuplicate.cpp -o containsDuplicate
./containsDuplicate
```
